Concept
・dharma is very simple browser that has a different purpose and concept. MobileSafari has a lot of feature, tab browsing and background processing, however, use too much physical memory.
・dharma is very light web browser and doesn't use physical memory very much. And the other funcation is that user can obtain bookmarks from social bookmark service, such as delicious and "はてな".

Main features(Image is changed when mouse over each title.)
・Bookmark
　→Has basic bookmark functions.
　→Such as, frequently visited sites.
・History
・Keyword, URL suggestion
　→Suggest keyword and URL based on your input history.
・Social bookmark
　→You can use social bookmark service, delicious and "はてな".
・Post a bookmark to delicious
