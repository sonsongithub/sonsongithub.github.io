MySQL/Ruby，Ruby/MySQLを使ってDBから大きいサイズのファイルをselectしてくると遅い．
めちゃくちゃ遅い．
RDBでバイナリを扱うなという話もあるけど，遅すぎてムカつくので，対策を探していたところ，Ruby-DBIで解決．
参考サイトは以下．
・配布元
・http://www.jiubao.org/ruby-dbi/ruby-dbi.html
・http://d.hatena.ne.jp/peregrination/20070925/1190711719
dbi-0.2.0.tar.gzを落としてきて，さっくり，インスコ．

ruby ./setup config --with=dbi,mysql
ruby ./setup setup
ruby ./setup install

これで終了．
