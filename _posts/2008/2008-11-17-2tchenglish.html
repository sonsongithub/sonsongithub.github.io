Features
・2tch is 2ch, "2ちゃんねる" is Japanese BBS, viewer which is implemented with iPhone SDK.
・2tch will make users navigate 2ch more comfortably than Safari.
・Main features
　- Smooth scrolling
　- Caching thread data
　- Searching thread titles with words
　- Writing, default setting "sage"
　- Popping up an image view via link to image URL
　- Popping up an comment view via anchor for a particular response.
　- Deleting cache files
　- Bookmark
　- History
　- Saving an image file into Photo storage of iPhone/iPod touch
