SSII2011のチュートリアル講演で紹介するサンプルコードの一覧です．

  • Quartz Help Library
  • Real time image processing framework for iOS
  • iOS SDK Hacks
  • CoreAR

Quartz Help Library

Quartz Help Libraryは，UIImageやCGImageなどのiOSでよく使う画像データと生のピクセルデータを変換するためのヘルプライブラリです．

Real time image processing framework for iOS

Real time image processing framework for iOSは，iOSでリアルタイム画像処理を行うためのフレームワークです．上のスクリーンショットのようにカメラ画像をリアルタイムに二値化するコードなどが含まれます．

iOS SDK Hacks, Chap.6

iOS SDK Hacks 第６章に記載のに，Accelerate.frameworkによるベンチマークのためのソースコードがあります．

CoreAR

CoreARというARToolKitライクな2次元コード認識フレームワークもあります．

改訂版資料

SSII2011チュートリアル講演で使用した資料のPDFです．容量などの都合で，ムービーは含みません．

外部リンク

Apple Developer connectionに登録後，Log inからログインして，iTunesからソースコードやセッションのビデオをダウンロードすることができます．カメラに関するコンテンツがいくつかあります．
