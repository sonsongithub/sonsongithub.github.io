How to use StoreSales
Step. 1 Download and install StoreSales for MacOSX
You need to couple your iPhone and your Mac for sending sales log to your iPhone.
At first, download StoreSales for MacOSX and install it in your Mac.


Download

Double click "StoreSales.dmg" icon.
Image file will be mounted and then following window will be opened.
And drug and drop "StoreSales.app" into "Application" folder.

Step. 2 Choose log folder and copy log files into the folder.
Next, run StoreSales. Set up StoreSales's log file folder. This folder is used to save log file obtained from iTunes connect. The default path is "/Users//Library/Application Support/StoreSales/log/". Copy sales log file obtained from iTunes connect into this folder.

Step. 3 Pairing, iPhone
Run StoreSales for iPhone, and let your iPhone and Mac join to the same WiFi network.
Move to "sync view", and tap "Pairing with Your Mac", then 4 digitis passcode is shown.

Step. 4 Pairing, MacOSX
Right now, change into your Mac and pull down "StoreSales for Mac", from system menuber, select "Pairing with new device".
Then, "passcode input window" is shown. Please input 4 digits passcode which is shown at iPhone. If the passcode you inputed is right, pairing is successed.



Step. 5 Receive sales log files from Mac
If pairing iPhone with Mac is succeeded, "Receive sales log from Mac" button is appeared. Push this button and then receiving sales log files from Mac will start.

Step. 6 Request confirmation
On MacOSX, if iPhone request Mac to send sales log files, a confirmation sheet is appeared. Push OK after confirmation iPhone's name if you accept the request, and then iPhone and Mac start to transfer them.
