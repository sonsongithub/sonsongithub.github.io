まず，インストールする．

>sudo port install mecab
>sudo port install mecab-ipadic-utf8

そして，このままだと，辞書なしで形態素解析してしまうので，
辞書を設定する．

sudo emacs /opt/local/etc/mecabrc

で，

dicdir = /opt/local/lib/mecab/dic/ipadic-utf8

という風に辞書のパスを追記しておく．
