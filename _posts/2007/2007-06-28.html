Javascript．
連想配列やら，いろいろなことができ便利なはずなのに，まともに勉強したことがない．
そこで，Javascriptのお勉強を胸にJavascriptでオブジェクト指向プログラミングをやってみたい．
とはいっても，知識がないので，いろいろなサイトを見て，勉強した．
ほとんど，そこいらのサル真似だけど，それがプログラミングの上達の道だからいたし方ない．
なるべく，僕が好きなC++とかの概念と結びつけながら感想，勉強のログを書き綴ってみる．
基本的に間違いだらけ・・・だと・・・思いますぜ．諸兄．
※クラス

function ClassA() {
this.prop = 1;
this.hello = function () {
alert( "Hello ClassA" );
}
}
objA = new ClassA();
objA.hello();

これは，C++でいうと．こんな感じ？

class ClassA {
public
int prop;
void hello();
}
ClassA::ClassA(){
prop = 1;
}
void ClassA::hello(){
alert( "Hello ClassA" );
}
objA = new ClassA;
objA->hello();

Javascriptは連想配列でもプロパティを呼べるのがいけてる．

alert( objA.prop );
alert( objA[ 'prop' ] );

※継承
継承は，prototypeという機能？を使う．
prototypeとは，そのオブジェクトの元となるモデル？みたいなものと理解した．

function ClassA() {
this.prop = 1;
this.hello = function () {
alert( "Hello ClassA" );
}
}
function SubClassA() {
this.hello = function () {
alert( this.prop );
}
}
SubClassA.prototype = new ClassA();
